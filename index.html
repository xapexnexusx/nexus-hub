Terminal Environment
gh repo clone xapexnexusx/nexus-dotfiles && cd nexus-dotfiles && ./install.sh && cd ..
Gateway + NEXUS COMMAND
gh repo clone xapexnexusx/nexus-gateway && cd nexus-gateway && ./install.sh && cd ..
